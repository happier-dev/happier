Get Started in Minutes

Set up Happier and start coding from anywhere in just four simple steps.

1

Download the App

Get Happier from the App Store or Google Play on your mobile device.

2

Install the CLI

Run npm i -g @happier-dev/cli on your computer.

3

Pair Your Device

Scan the QR code in the app to securely connect your phone to your computer.

4

Start Coding

Run happier instead of claude and code from anywhere!

Terminal
# Instead of running claude directly:
$ claude
# Use happier to enable remote access:
$ happier
# Or for other providers:
$ happier codex
$ happier opencode
$ happier gemini
Security First

Your Code is Your Code. Private. Always.

Happier is designed with privacy as a foundation, not an afterthought. We use the same encryption technology trusted by Signal.

End-to-End Encryption

Built using TweetNaCl, the same cryptographic library used by Signal. Your data is encrypted before it leaves your device.

Zero-Knowledge Architecture

Servers cannot read your data. Encryption keys never leave your devices. We literally cannot access your code.

Built in Switzerland

Developed in Switzerland with a strong focus on data protection, privacy laws, and developer transparency.

Built by Developers, for Developers

Happier is completely open-source. We built this because we wanted a more powerful, more social way to interact with AI agents — and we want to build it in the open, shaped by the people who actually use it.

Open Development

Transparent discussions, public roadmap, and visible progress. See exactly what we're building.

Fast Feedback Loops

Quick responses on issues and pull requests. We iterate rapidly based on real user needs.

Real Developer Pain

Focus on solving actual problems, not chasing hype. Every feature has a real use case behind it.
